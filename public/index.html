Product Overview

它是做什么的？

它不是交易软件，是一个每日复盘助手：把用户持仓、大盘变化、行业新闻、公司消息和 AI 校验整理成一份中文归因日报。

01

用户输入

用户提供模型 API Key 和持仓列表。

02

数据采集

Phase 0 板块脉搏 + Phase 1 价格新闻 + Phase 2 深搜。

03

AI 分析

Executor 生成报告 → Verifier 逐条审查 → 不通过则自动修正（最多 3 轮）。

04

日报输出

大盘归因、板块脉搏、个股卡片、跨股关联、来源链接、数据完整性。
